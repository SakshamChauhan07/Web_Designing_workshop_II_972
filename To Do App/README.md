# TaskFlow To-Do App

TaskFlow is a simple React to-do app made with HTML, CSS, JavaScript, and React.

## Features

- Add new tasks
- Choose task category
- Set task priority
- Mark tasks as done or pending
- Delete tasks
- Filter all, pending, and done tasks
- Clear completed tasks
- View live task counters

## How to Run

Open `index.html` directly in a browser.

You can also run a local server with Node.js:

```bash
node server.js
```

Then open:

```text
http://127.0.0.1:8000/index.html
```

## Files

- `index.html` - main HTML file
- `styles.css` - app styling
- `app.js` - React app logic
- `server.js` - optional local server
